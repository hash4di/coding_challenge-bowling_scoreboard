# Coding Challenge: Bowling Scoreboard

## Tech stack:
  - `Ruby: 2.7.0`
  - `Rails: 6.0.2.1`
  - `Bundler: 2.1.4`
  - `Rspec: 4-0-dev [issues with new Rails 6 build, stil WIP]`

## Installation:
  <pre>
    $ gem install bundler
    $ bundle install
  </pre>
  
## Runing:
  <pre>
    1. $ bunde exec bin/rails s
    2. open browser on http://localhost:3000
  </pre>
  
## Testing:
  <pre>
    $ bunde exec rspec
  </pre>
  
## Rules:
